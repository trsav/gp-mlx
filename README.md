[![Actions Status][actions-badge]][actions-link]

Lightweight Fast Gaussian Process Implementation in MLX

## Installation

```bash
python -m pip install gpmlx
```

From source:
```bash
git clone https://github.com/trsav/gp-mlx
cd gp-mlx
python -m pip install .
```
